# Sebi's Sight-Reader

A kid-friendly music sight-reading web app that listens for notes and gives instant feedback.

## Run locally

1. Install dependencies:

```
pnpm install
```

2. Start the dev server:

```
pnpm dev
```

3. Open the URL shown in the terminal (typically `http://localhost:5173`).

## Tests

```
pnpm test
```

## Build

```
pnpm build
```

## Preview the build

```
pnpm preview
```
